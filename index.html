1. Init:

    Para que sirve? Es el primer comando que se debe ingresar en la consola, para dar por inicializacion a git.

  2. Add:

    Para que sirve? El add sirve para añadir archivos al repositorio local gracias a instrucciones como git add --all.

  3. Commit:

    Para que sirve? Para empacar todos los cambios hechos a los archivos y dejarlos listos con su respectiva descripcion para subir al repositorio remoto.

  4. Push:

    Para que sirve? El push sirve para subir los archivos ya listos del repositorio local, donde previamente se hizo el commit.

  5. Pull:

    Para que sirve? El pull sirve para "agarrar" todos los archivos del repositorio remoto y traerlos al repositorio local.

  6. Clone:

    Para que sirve? El clone sirve para clonar el repositorio remoto en mi repositorio local, pudiendo acceder a todos los archivos.

  7. Remote Add:

    Para que sirve? Para anadir un repositorio local a nuestro computador desde 0.
